Introduction

Created out of a mixture of frustration and utility the Valor Admin was created to speed development and reduce the amount of repetition for our projects.

With a large amount of common scenarios our hope is that this generic admin will continue to grow as we work to encapsulate different scenarios we commonly encounter.

The Valor Admin was created with Angular 12, Nest 7, Prisma, ngx-bootstrap 7, and Bootstrap 5

Note: This is the Alpha version of the Valor Admin and may have some quirks. Make sure to drop a comment on our Public GitHub if you run into any issues or see any areas for improvement.
